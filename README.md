## ✨ Features
 
- **4 time modes** — 15s · 30s · 1 min · 2 min
- **Live WPM tracking** during the test
- **Accuracy score** on results screen
- **Animated cursor** with real-time correct / incorrect character highlighting
- **Auto-scroll** — words smoothly scroll as you progress
- **Keyboard-first** — `tab` restarts instantly, no mouse needed
- Minimal UI with zero distractions

## 🚀 Getting Started
 
### Prerequisites
 
- [Node.js](https://nodejs.org) v18+
- npm or yarn
 
### Installation
 
```bash
# Clone the repo
git clone https://github.com/your-username/typemaster.git
cd typemaster
 
# Install dependencies
npm install
 
# Start the dev server
npm run dev
```
 
Then open [http://localhost:5173](http://localhost:5173) in your browser.
 
### Build for Production
 
```bash
npm run build
npm run preview
```

## 🎮 How to Use
 
| Key | Action |
|-----|--------|
| **Start typing** | Test begins automatically |
| `Space` | Move to next word |
| `Backspace` | Delete last character |
| `Tab` | Restart test instantly |
| `Tab` + `Enter` | Restart from results screen |

## 🎨 Design
 
Inspired by [MonkeyType](https://monkeytype.com). Built with a flat, distraction-free aesthetic:
 
| Token | Value |
|-------|-------|
| Background | `#2c2e31` |
| Accent (yellow) | `#e2b714` |
| Correct text | `#d1d0c5` |
| Untyped text | `#646669` |
| Error text | `#ca4754` |
| Font | Roboto Mono 300/400 |
